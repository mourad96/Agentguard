# Runtime Verification

This directory contains the runtime verification engine for AgentGuard.

## Overview

The Runtime Verification module implements the core model checking and verification logic. It dynamically learns system behavior and verifies agent actions against safety thresholds.

## Key Features

- Dynamic model learning and state space construction
- Incremental verification (e.g., every 5 iterations or transactions)
- Safety threshold validation
- Path analysis and preemptive termination
